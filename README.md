# 💇‍♀️ Studio Aris - Frontend (Versión Local)

Aplicación frontend desarrollada con Angular para la gestión de un estudio de belleza. Esta versión corresponde al desarrollo inicial del proyecto ejecutado en entorno local.

---

## 🚀 Tecnologías

* Angular
* TypeScript
* HTML
* CSS
* RxJS

---

## 📌 Descripción

Proyecto enfocado en la construcción de una interfaz web moderna para la administración de:

* Usuarios
* Citas
* Ventas
* Productos y servicios

Incluye integración con API REST y manejo de autenticación mediante JWT.

---

## ⚙️ Ejecución

```bash
npm install
ng serve
```

Abrir en navegador:

```
http://localhost:4200
```
